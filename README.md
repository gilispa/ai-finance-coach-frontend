# AI Finance Coach – Frontend

This is the frontend for **AI Finance Coach**, a personal finance tracking application.

The app allows users to:
- Register incomes and expenses
- Categorize spending
- View summaries and recent activity
- Receive AI-powered financial insights (via backend API)

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- Fetch API
